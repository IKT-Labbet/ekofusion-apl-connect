# Ekofusion-APL-connect
Ekofusion Inkubator APL Connect kopplar samman skolor med företag och organisationer som kan erbjuda APL- eller LIA-platser. Projektmål: Bygga en fullstack-applikation där företag kan registrera intresse för att ta emot APL-elever eller LIA-studenter i framtiden, och där skolor kan söka efter passande företag.
